<h1 align="center">To-Do List Go & React </h1>

<p align="left">
This is a simple To-Do List application developed using the following technologies:

- Backend: Golang
- Frontend: Vite + React.js (TSX)
- Database: MongoDB
- UI Library: Chakra UI

</p>
